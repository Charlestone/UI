Problemas encontrados:

Hemos intentado hacer animaciones verticales, pero siempre las cartas se nos escondían hacia la esquina superior izquierda.

A veces, al cargar la web, firefox muestra un error porque las fuentes no se han cargado.


Decisiones:

Hemos decidido que el boton de start se esconde hasta que se pulsa el boton reset para que el usuario no intente empezar otra partida cuando haya una en pantalla.

Hemos decidido que las animaciones sean simples porque creemos que el mero hecho de que haya animación demuestra que tenemos el conocimiento suficiente para completar el ejercicio. No obstante, somos conscientes de que otra animación más compleja haría más atractiva la web (de igual modo con la disposición de las cartas).
